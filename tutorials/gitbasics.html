Git Basics

First configuration

git config --global user.name "John Smith" git config --global user.email "johnsmith@johnson.com"
SSH Keys

Type in the following command to see if any existing SSH keys are present:

ls -al ~/.ssh

If not:

ssh-keygen -t ed25519 -C "johnsmith@johnson.com"

Check if SSH Agent is running

eval "$(ssh-agent -s)"

Add your SSH private key to the ssh-agent. If you created your key with a different name, or if you are adding an existing key that has a different name, replace id_ed25519 in the command with the name of your private key file.

ssh-add ~/.ssh/id_ed25519

Copy the contens of the id_ed25519.pub file to your clipboard

clip < ~/.ssh/id_ed25519.pub

Go to github profile picture / Settings / Access / SSH and GPG keys / New SSH Key

Add a descriptive title like "Lenovo Notebook"

Paste your key into the “Key” field.

You are done!
